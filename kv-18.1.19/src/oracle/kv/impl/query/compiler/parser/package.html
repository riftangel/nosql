A number of files in this package are generated by Antlr v4 using the generate-ddl target: ant generate-ddl The generated files include TableBaseListener.java TableLexer.java TableListener.java TableParser.java By default the generated files result in Eclipse errors due to unused imports. When the grammar file, Table.g, is modified in a way that requires regeneration take these steps: 1. ant generate-ddl 2. test changes 3. using Eclipse fix the warnings in the generated files 4. push new code
